## Alias

| command alias           | execute comand         |
| ----------------------- | ---------------------- |
| ..                      | `cd ../`               |
| ...                     | `cd ../../`            |
| ....                    | `cd ../../../`         |
| ls                      | `eza`                  |
| tree                    | `eza --tree`           |
| cat                     | `bat`                  |
| find                    | `fd`                   |
| ps                      | `procs`                |
| man                     | `tldr`                 |
| diff                    | `delta --side-by-side` |
| neofetch                | `fastfetch`            |
| hm (for not NixOS user) | `home-manager switch`  |
| gc                      | `nix-collect-garbage`  |
